# 📱 Social Post Prompts

Para crear posts para LinkedIn, Instagram y Facebook.

---

## Prompt: Generate Post

```
Crea un post de social media con estas especificaciones:

**Red:** [LinkedIn / Instagram / Facebook grupos]
**Idioma:** [Español / Inglés]
**Producto o tema:** [Tu producto / marca personal / tema de tu industria]
**Tipo de post:** [Educativo / Before-after / Social proof / Pregunta / Oferta / Historia]
**Audiencia:** [Describe tu audiencia — Ej: Latinos en USA / Solopreneurs / Dueños de negocios]
**Objetivo:** [Conseguir DMs / Aumentar seguidores / Generar leads / Brand awareness]

Usa mi voz:
- [Ej: Directo y sin fluff]
- [Ej: Casual pero inteligente]
- [Ej: Sin jerga corporativa]
- [Ej: Muestra resultados concretos]
- [Ej: Párrafos de 1-2 líneas máximo]
- [Ej: Máx N emojis si aplican naturalmente]

(Ver skills/your-voice.md para instrucciones completas de voz)
```

## Templates by Type

<!-- FILL IN: Adapta estos templates a tu voz y productos. -->

### Before/After
```
Antes: [Situación del cliente antes de tu solución — específica y cuantificada].

Después: [Resultado concreto después].

Esto es lo que hace [tu solución/automatización] real.

¿[Pregunta que invita a la reflexión]?
```

### Educational Hook (inglés)
```
Most [tu audiencia] [problema que tienen — con dato o porcentaje si tienes].

Here's the fix: [solución concreta en 1 línea]

[Explicación en 2-3 bullets cortos]

[Cierre: insight o CTA suave]
```

### Social Proof
```
[Tipo de cliente] from [ciudad/área] was spending [tiempo/dinero] on [problema] every day.

[Lo que hicimos juntos — 1-2 líneas].

If you're doing the same thing manually, let's talk.
```

### Engagement Question (español)
```
Pregunta rápida para [tu audiencia]:

¿[Pregunta relevante para tu negocio]?

👍 [Opción A]
💬 [Opción B]
❌ [Opción C]

[1 línea de por qué importa la respuesta]
```

### Direct Offer
```
Looking for [tipo de cliente que quieres] who want to [resultado específico].

We [cómo lo logramos en 1 línea].

[CTA — Ej: First X people to DM me get Y. / Reply here if interested.]
```
