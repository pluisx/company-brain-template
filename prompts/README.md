# 💬 Prompts

Prompts listos para usar. Copia, pega, ajusta el `[texto en corchetes]`, y ejecuta.

## Available Prompts

| File | When to Use |
|------|-------------|
| `cold-outreach.md` | Cold prospecting (LinkedIn, email, DM) |
| `follow-up.md` | Follow-up after first contact without response |
| `social-post.md` | Create posts for LinkedIn, Instagram, Facebook |
| `proposal.md` | Service proposal for a potential client |

## How to Use with Claude Code

```bash
# In Claude Code, say:
"Read prompts/cold-outreach.md and use it to write a message for [describe the prospect]"
```

## How to Use with ChatGPT/Claude web

1. Open the prompt file
2. Replace `[variables in brackets]` with your real info
3. Paste it in the chat and add extra context if needed

---

> 💡 These prompts work best when you've already filled in `context/ABOUT.md`, `context/COMPANIES.md`, and `skills/your-voice.md` — the AI will use that context automatically.
