# 🔄 Follow-up Prompts

Para seguimiento después de primer contacto sin respuesta, o para nutrir leads que no cerraron todavía.

---

## Prompt: Generate Follow-up

```
Necesito un follow-up para este prospecto:

**Canal original:** [Donde hiciste el primer contacto]
**Días desde el primer contacto:** [X días]
**Primer mensaje enviado:** [Pega el mensaje original aquí]
**¿Respondieron algo?** [Sí/No — si sí, qué dijeron]
**Producto:** [Tu producto]
**Objetivo:** [¿Quieres una llamada? ¿Demo? ¿Que prueben gratis?]

Reglas:
- UN solo follow-up máximo si no hay respuesta
- No seas ansioso ni insistente
- Agrega valor nuevo — no solo "¿viste mi mensaje?"
- Mantén la misma longitud o más corto que el original
- Usa la voz de [TU NOMBRE] — ver skills/your-voice.md
```

## Follow-up Templates

### Follow-up #1 — No response (3-5 days later)
```
Hey [Name], quería hacer un follow-up sobre [producto/oferta].

[Agrega algo nuevo: stat relevante, caso de estudio, o pregunta diferente]

[CTA específico]
```

### Follow-up for "We'll think about it" (1-2 weeks later)
```
Hey [Name], ¿tuviste chance de pensar en lo que hablamos sobre [producto]?

Solo quería saber si tienes preguntas o si hay algo que pueda clarificar.

[CTA suave — sin presión]
```

### Follow-up with new value (inglés)
```
Hey [Name], one thing I forgot to mention —

[Dato nuevo, feature relevante, o caso de éxito de cliente similar]

Still interested in connecting? [calendario]
```

### Re-activation after 45+ days
```
Hey [Name], sé que han pasado unas semanas.

¿Sigue siendo relevante [el problema que mencioné antes] para ti?

Si cambió el timing, sin problema — solo quería checar.
```
