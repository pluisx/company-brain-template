# 🤖 Agents

Instrucciones para agentes de IA especializados. Cada archivo es un "system prompt" que puedes copiar y pegar en Claude, ChatGPT, o cualquier plataforma.

## Available Agents

| Agent | What It Does | When to Use |
|-------|-------------|-------------|
| `sales-agent.md` | Prospecting, outreach, closing deals | When you need more clients |
| `content-agent.md` | Posts, copy, emails | When you need content |
| `research-agent.md` | Market research, competition, opportunities | When you need intelligence |

## How to Use

```
1. Open the agent file you need
2. Copy ALL the content
3. Paste it as the first message in ChatGPT/Claude
4. Then give the specific task
```

Or in Claude Code:
```
> "Read agents/sales-agent.md and act as that agent for [specific task]"
```
